Field Description Example
id The integer representation of the unique identifier for this Tweet. This number is very large and some programming languages may have difficulty/silent defects in interpreting it. Use id_str instead.
id_str The string representation of the unique identifier for this Tweet. Implementations should use this rather than the large integer in id
contributors An array of user ids (usually only one) indicating the user who authored the tweet on behalf of the official tweet author. "contributors": [ 819797 ]
annotations Currently unused. Future/beta home for status annotations.
text the actual text of the status update
